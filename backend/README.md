# 3-Tier API

A simple 3-tier architecture API for user authentication and system info, built with **Node.js**, **Express**, and **PostgreSQL**.

---

## Features

- User registration and login
- Profile retrieval and token verification
- Health check and API info endpoints
- PostgreSQL integration
- JSON-based REST API
- CORS enabled for cross-origin requests

---

## Requirements

- Node.js >= 18
- PostgreSQL >= 12
- npm or yarn
